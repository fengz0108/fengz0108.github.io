Elementary OS安装配置指南

Elementary OS简介与安装

Elementary OS是一款基于Ubuntu精心打磨美化的桌面Linux发行版，号称 “最美的Linux”。Elementary 最早是Ubuntu的一个美化主题项目，现在成了独立的发行版。我们安装的的版本是Elementary OS Luna系统，基于Ubuntu12.04打造下载链接。

安装过程不必多说了和Ubuntu一样，需要注意的是系统引导工具就装载/下即可，然后重启会进入Windows然后用easyBCD添加引导就好，这样是最方便的。

安装之后的配置

安装更新

安装时选择英语，安装完后进入系统，点左上角，找到system—>update,打开后，选择setting，选择最快的更新服务器。

打开终端：

sudo apt-get update
sudo apt-get upgrade

安装elementary-tweaks

sudo apt-add-repository ppa:versable/elementary-update
sudo apt-get update
sudo apt-get install elementary-tweaks

安装好了设置里面会多一个tweaks可以修改很多参数。

中文支持

System Settings ->Lauguage Support ->install/remove language 选择 chinese（sim）。安装完后，Apply System Wide 上面的语言栏 汉语在最下面，拖到最上面，点击Apply System Wide即可。

文件浏览器右键终端

新建一个文件terminal.contract在里面填写以下内容

[Contractor Entry]
Name=此处打开终端
Icon=terminal
Description=此处打开终端
MimeType=inode;application/x-sh;application/x-executable;
Exec=pantheon-terminal -d %U
Gettext-Domain=pantheon-terminal

再cp到/usr/share/contractor/terminal.contract

安装软件

  • 安装sogou

先删除ibus输入法

sudo apt-get remove --purge ibus
sudo apt-get autoremove

安装搜狗拼音

sudo add-apt-repository ppa:fcitx-team/nightly
sudo apt-get update
sudo apt-get install fcitx-sogoupinyin

在语言支持那里选择输入法fcitx。

安装完sogou输入法之后，待重启系统后，fcitx随系统启动。

右键托盘的fcixt输入法图标，配置，如果没有sogou输入法，请点加号添加即可。默认CTRL+SPACE切换输入法。

  • gimp和smplayer，以及build-essential

sudo apt-get install gimp smplayer build-essential

移除系统自带的播放器和浏览器

sudo apt-get remove totem midori-granite noise

  • 安装chrome

下载google-chrome-stable.deb包，百度网盘链接:64位包 密码: 0eob;
32位包 密码: 5nvr

安装

sudo dpkg -i google-chrome-stable*.deb

卸载

sudo dpkg -P google-chrome-stable

  • 安装WPS

1.wps官网下载wps for linux的deb安装包

2.安装配置环境

sudo apt-get update
sudo apt-get install libc6-i386
sudo apt-get install ia32-libs

3.安装过程中肯定会报错不用理会，直接安装deb安装包就好，然后还会提示缺少必要的包，这个时候修复依赖即可。

sudo apt-get install -f
